# Offroad Frontier

Офф-роад симулятор с продвинутой физикой транспорта и деформируемым ландшафтом.

## Описание

Offroad Frontier - это игра про преодоление бездорожья на грузовиках 4x4. 
Вас ждут грязь, снег, реки и сложные маршруты. Используйте лебёдку, 
блокировки дифференциала и полный привод для достижения цели.

## Особенности

- **Реалистичная физика подвески** - каждое колесо имеет независимую подвеску
- **Деформируемый ландшафт** - колея от колёс остаётся на мягких поверхностях
- **Динамическая погода** - дождь превращает грязь в болото, снег заметает дороги
- **Система поверхностей** - разное сцепление для асфальта, грязи, снега, льда
- **Лебёдка** - крепите трос к деревьям и камням для вытягивания
- **Несколько камер** - вид от третьего лица, из кабины, на капот
- **Мини-карта** - навигация по пересечённой местности

## Управление

| Клавиша | Действие |
|---------|----------|
| W / ↑ | Газ |
| S / ↓ | Тормоз / Задний ход |
| A / ← | Поворот влево |
| D / → | Поворот вправо |
| Пробел | Ручной тормоз |
| C | Смена камеры |
| E | Лебёдка (вытянуть/втянуть) |
| Q | Блокировка дифференциала |
| F | Включить/выключить 4WD |
| M | Мини-карта |
| Esc | Меню паузы |

## Требования

- Rust 1.70+
- Движок RTGC-0.7 (https://github.com/NAREZKA578/RTGC-0.7)

## Сборка

```bash
# Клонируйте репозиторий
git clone <repository_url>
cd offroad_frontier

# Убедитесь, что RTGC-0.7 находится в соседней директории
# ../RTGC-0.7

# Сборка
cargo build --release

# Запуск
cargo run --release
```

## Зависимости

- `rtgc` - игровой движок
- `nalgebra` / `glam` - математические операции
- `serde` - сериализация настроек
- `noise` - генерация ландшафта
- `winit` / `wgpu` - окно и рендеринг
